# Don's Portfolio Site

## About

This is the source code for my portfolio site. The site is built using [Jekyll](https://jekyllrb.com/). 


## Developing

Clone this repo. Run the following on terminal:
```
gem install bundler
bundle install
bundle exec jekyll serve
```
